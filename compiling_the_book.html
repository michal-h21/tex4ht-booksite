Chapter 3
Compiling the Book

You can compile the output files using any standard LaTeX distribution which includes TeX4ht. 
$ make4ht -e build.lua -c config.cfg -lj index -d out document.tex

The -l option ensures that LuaLaTeX is used as the compiler.
