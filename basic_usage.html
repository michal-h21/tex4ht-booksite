Chapter 2
Basic Usage

2.1 Create a New Github Repository

To start your own online book using this template, click the "Use this template" button at the top of the GitHub repository page. This will create a new repository in your account with the same structure and files. You can then clone it, customize the content, and start building your own slides and handouts.
